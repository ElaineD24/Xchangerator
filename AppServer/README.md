#Xchangerator AppServer


## For development:
Create `.env` file with key value pairs for environment variables needed by the server

Sample `.env`
```
NODE_ENV=development
PORT=3000
```

launch
```
npm run dev
```

## For production

Sample `.env`
```
NODE_ENV=production
```

launch
```
npm start
```

reload
```
npm run reload
```
