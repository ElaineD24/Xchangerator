# Xchangerator AppServer


## Development:
Create `.env` file with key-value pairs for environment variables used by the server

Sample `.env`
```
NODE_ENV=development
API_KEY=xxx
PORT=3000
```

launch
```
npm run dev
```

## Production

Sample `.env`
```
NODE_ENV=production
API_KEY=xxx
```

launch
```
npm start
```

reload
```
npm run reload
```
